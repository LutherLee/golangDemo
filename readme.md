# Go Fiber RestApi Example + Basic Endpoint testing
<br/>

## Install assertion library dependencies
- [Testify](https://github.com/stretchr/testify)

Type in terminal/command prompt
```
go get github.com/stretchr/testify
```

## To run the server
Type in terminal/command prompt
```
go run main.go
```

## To run test 
Type in terminal/command prompt
```
go test
```
